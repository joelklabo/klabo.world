# Aesthetic Directions

Commit to ONE direction and execute with precision.

## Direction Archetypes

### Brutally Minimal
- Maximum negative space
- Single accent color
- Ultra-thin or ultra-bold type
- No decorative elements
- **Example:** Apple product pages

### Editorial/Magazine
- Strong typography hierarchy
- Pull quotes and large numbers
- Column-based layouts
- Art-directed image placement
- **Example:** NYT feature articles

### Retro-Futuristic
- Monospace fonts
- Terminal/command aesthetics
- Scan lines, glitch effects
- Neon on dark backgrounds
- **Example:** cyberpunk interfaces

### Organic/Natural
- Soft curves and blobs
- Earth tone palettes
- Hand-drawn elements
- Irregular shapes
- **Example:** wellness apps

### Industrial/Utilitarian
- Exposed structure
- Mono fonts, dense information
- Minimal color (grays + one accent)
- Grid-locked layouts
- **Example:** GitHub

### Playful/Toy-like
- Rounded everything
- Bright, saturated colors
- Bouncy animations
- Oversized elements
- **Example:** Notion

### Luxury/Refined
- Thin serif typography
- Lots of breathing room
- Subtle animations
- Muted, sophisticated palette
- **Example:** fashion brands

## Applying to klabo.world

Current aesthetic: **Editorial/Industrial hybrid**
- Dark background (`zinc-900`)
- Amber accent (`amber-500`)
- Code/technical content focus
- Blog-first layout

### Consistent Choices
| Element | Pattern |
|---------|---------|
| Headings | Large, bold, amber underlines |
| Code blocks | Dark with amber accents |
| Callouts | `bg-amber-500/10 border-l-4 border-amber-500` |
| Cards | `bg-zinc-900/50 border border-zinc-800` |
| Links | Amber with underline on hover |
